Battlebus

Battlebus is an application that allows users to become familiar with their local, state, and federal representatives. The application prompts the user to create an account, and with information provided by the user, displays their politicians. The user can then save politicians to their profile along with notes as they get to know the representative. The goal of this app is to increase voter education, particularly concerning local representatives who tend to have less notoriety. Battlebus is built using JavaScript, HTML, CSS, ReactJS and Semantic UI.

Prosé

Prosé is intended for book clubs as a tool to ease the process of selecting the next book and selecting a book that the majority of members are interested in. The user can belong to one or more clubs, and each club has a dashboard that features the book that the club is currently reading. Users can search for books under the club's "Find Books" tab, and they can choose to suggest a book from that list to their fellow members. Members can then vote on books and they appear in descending order by number of votes. The app is created using C#, ASP.NET Core, SQL Server, Bootstrap, and the Google Books API.
